\subsection{Tổng quan thư mục}
Bài lab được tổ chức theo các thư mục sau:
\begin{itemize}
    \item \textbf{1\_RTSP\_RTP}: Chứa source code cho tiêu chí 1.
    \item \textbf{2\_HD}: Chứa source code cho tiêu chí 2.
    \item \textbf{3\_CACHE}: Chứa source code cho tiêu chí 3.
    \item \textbf{4\_REPORT}: Chứa báo cáo của bài lab (tiêu chí 4).
    \item \textbf{video}: Chứa các video với các độ phân giải khác nhau phù hợp với từng tiêu chí.
\end{itemize}
Phần source code cũng đã được up lên \href{https://github.com/quocnam612/socket-programming}{Github}.
