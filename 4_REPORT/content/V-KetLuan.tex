\section{Kết luận}
\subsection{Kết quả đạt được}
Cả hai phương pháp đều đáp ứng tốt yêu cầu thử nghiệm mô hình LLM trên C++ như:
\begin{itemize}
    \item Gửi câu hỏi từ C++ đến mô hình LLM và nhận được câu trả lời chính xác.
    \item Kích hoạt mô hình cục bộ mà không phụ thuộc vào kết nối Internet.
    \item Tạo được  module autoAsk cho phép người dùng nhập câu hỏi liên tục theo dạng hội thoại đơn giản.
\end{itemize}

\subsection{Hướng phát triển}
Chương trình có thể được mở rộng theo nhiều hướng nhằm nâng cao tính ứng dụng và độ hoàn thiện. Một số hướng phát triển khả thi bao gồm:
\begin{itemize}
    \item Tích hợp REST API hoàn chỉnh
    \item Giao diện người dùng (GUI)
    \item Triển khai vào ứng dụng thực tế
\end{itemize}
